# Dialexa Javascript Style Guide

## How To Use

```
    npm install eslint --save-dev
    npm install eslint-config-dialexa --save-dev
```

### .eslintrc

```
    {
        "extends": "dialexa"
    }
```

### ES6

```
    {
        "extends": "dialexa/es6"
    }
```

### Ember
```
    {
        "extends": "dialexa/ember"
    }
```
